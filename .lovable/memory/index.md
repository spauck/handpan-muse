# Project Memory

## Core
Bun is the mandatory package manager. Biome for linting/formatting (2-space, double quotes).
Primary state lives in URL query params. Auto-save to localStorage every 3s.
Three-hand system (Right, Left, Any). Neutral hand auto-adjusts HSL for contrast in light/dark mode.
Fluid single-row CSS grid layout. Beat cells strictly maintain a 7:9 aspect ratio.
Composition is a flat array of Bars; each bar has its own length and a `breakBefore` flag that begins a new row.
